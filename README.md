# ground-making
マインクラフト用データパックのテストリポジトリです。

## インストール

[リリースページ](https://github.com/akihamitsuki/ground-making/releases)からzipをダウンロード。

`minecraft/saves/[ワールド名]/datapacks`にzipをそのまま配置。


## 使い方

データパックを入れたワールド内から次のコマンドで実行できます。
```
/function [関数名]
```

使える関数名は以下を参照。

## 関数名

`grass:flat_ground`
: 草ブロックで地面を均します。
